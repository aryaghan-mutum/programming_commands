#### Tabs 

- `tabnew filename`    Open a file in a new tab 
- `tabn`  		       Switch to new tab
- `tabp`  		       Switch to prev tab
- `tabc` 		       Close all the windows except the last tab
- `bf`			       Go to first file
- `bl`			       Go to last file
- `bn`			       Go to new file
- `bp`			       Got to prev file
- `bw`                 Close file
