				### VIM Cheat Sheet ###



#### Status command 

- `Ctrl-G`          Shows the file name, line number of the cursor, total lines


#### Screen Movement 

- `Ctrl-e` 		       Move screen down one line 
- `Ctrl-y` 		       Move screen up one line 
- `Ctrl-b`             Move back on full screen




#### Calculator
- `<Ctrl-r>=` 100/50
