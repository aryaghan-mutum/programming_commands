#### Save

| Commands 		| Description 				                     		     | Lisp procedure 	 | 
|---------------|------------------------------------------------------------|-------------------|
| `C-x C-s`     | save a file                  							     |					 |
| `C-x s`       | save all files                  							 |					 |
| `C-s C-w`     | saves the file with a different name, asks for the name.   |					 | 
| `M-x`		    | recovers the auto saved file                               |                   |
