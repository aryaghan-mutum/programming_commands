#### Exit

| Commands 		| Description 				                     		     | Lisp procedure 	 | 
|---------------|------------------------------------------------------------|-------------------|
|`C-z`          | exit and enter emacs  								     |                   |
|`C-x C-c`      | exit emacs perpetually									 |					 |
|`C-g`          | quit the current Commands                                  |					 |
|`C-x o C-x 0`  | close the current file 									 |                   |
