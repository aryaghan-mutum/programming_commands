#### Exit

| Commands 		| Description 				                     		     | Lisp procedure 	 			| 
|---------------|------------------------------------------------------------|------------------------------|
|`C-z`          | exit and enter emacs  								     |`(suspend-frame)`             |
|`C-x C-c`      | exit emacs perpetually									 |`(save-buffers-kill-terminal)`|				 
|`C-g`          | quit the current command                                   |					 			|
