#### Mode

| Commands 				 | Description 				                     		      | Lisp procedure 	                      | 
|------------------------|------------------------------------------------------------|---------------------------------------|
| `C-x` `C-q`  			 | switch on/off read only mode aka: `M-x read-only-mode`     | `(toggle-read-only)`                  |
| `M-x *-mode <tab>`     | display all modes										  |										  |
| `M-x describe-mode`    | describe modes 											  | `(describe-mode)` 					  |
