#### Buffer

| Commands 		| Description 				                     		     | Lisp procedure 	   | 
|---------------|------------------------------------------------------------|---------------------|
| `C-x C-b`     | list all buffers (shows buffer & file name of the buffer	 |`(list-buffers)`	   |
| `C-x b`       | open specific buffer										 |`(switch-to-buffer)` |
