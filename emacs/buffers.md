#### Buffer

| Commands 		| Description 				                     		     | Lisp procedure 	 | 
|---------------|------------------------------------------------------------|-------------------|
| `C-x C-b`     | list all buffers											 |					 |
| `C-x b`       | specific buffer											 |					 |
| `C-x h`       | marks from the beginning to the end of the buffers         |                   |
