#### Search

| Commands 		| Description 				                     		        | Lisp procedure 	| 
|---------------|---------------------------------------------------------------|-------------------|
| `C-s`         | forward search                    						    |					|
| `C-r`         | backward search                 								|					|
| `C-M-x`       | regex search                    								|		            |					 
| `C-x C-f`     | search a file. If the file doesn't exist, it creates the file |                	|	
| `C-x i`       | insert a file                   								|		            |					 
| `M-x grep`    | greps a pattern in the files                   				|				    |					 
| `M-x rgrep`   | recursively grep in a dir                  					|					|
