#### Editing

##### cut-copy-paste 
- `C-w`     Cut
- `C-k`     Cuts the entire line including spaces and saves in buffer
- `M-w`     Copy
- `C-y`     Paste

##### undo-redo 
- `C-/`      Undo
- `C-g C-/`  Redo

##### inc-dec texts
- `C-x C-+` Increase text size
- `C-x C-+` Decrease text size

##### select a region 
- `C-x h`   Select all in the file
- `M-h`	    Select a paragraph
